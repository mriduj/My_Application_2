# My Application 2

## Overview

My Application 2 is a simple Android application developed using Kotlin and XML in Android Studio. The application demonstrates event handling by updating the text displayed on the screen when a button is clicked.

## Features

* Simple and user-friendly interface
* Displays a TextView with an initial message
* Includes a clickable button
* Updates the TextView dynamically when the button is pressed

## Technologies Used

* Kotlin
* Android Studio
* Android SDK
* XML Layouts

## How It Works

1. The application starts by displaying the text **"Hello Android"**.
2. The user clicks the **"Click Me"** button.
3. The TextView updates to display **"Button Clicked"**.

### Example

Initial Output:
Hello Android

After Button Click:
Button Clicked

## Project Structure

* `MainActivity.kt` – Handles the button click event and updates the TextView.
* `activity_main.xml` – Defines the user interface containing a TextView and a Button.

## Learning Objectives

This project was created to understand:

* Android UI components
* Working with TextView and Button
* Handling button click events
* Updating UI elements programmatically using Kotlin
* Basic Android application development

## How to Run

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle files.
4. Run the application on an Android emulator or physical device.
